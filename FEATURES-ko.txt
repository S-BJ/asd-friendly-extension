ASD-Friendly UI 기능 설명

[기본 제어와 시작 설정]
-전체 켜기/끄기: 확장 프로그램의 모든 보조 기능을 한 번에 켜거나 끌 수 있어요.
-첫 실행 설정: 처음 사용할 때 편안한 기본 설정을 빠르게 고를 수 있어요.
-언어 설정: 팝업 화면 언어를 자동, 영어, 한국어 중에서 선택할 수 있어요.
-편안함 프리셋: 최소 설정, 부드러운 밝은 화면, 부드러운 어두운 화면, 글 읽기 중심, 움직임 최소화, ADHD 집중 같은 묶음 설정을 적용할 수 있어요.
-활성 상태 표시: 현재 페이지에서 확장 기능이 적용 중인지 작은 표시로 알려줘요.
-빠른 토글 버튼: 페이지 위에 떠 있는 작은 버튼으로 자주 쓰는 기능을 빠르게 켜고 끌 수 있어요. (선택)
-키보드 단축키: AI 분석, 확장 켜기/끄기, 포커스 스포트라이트, 리더 모드를 키보드 단축키로 실행할 수 있어요. (브라우저 설정에서 변경 가능)

[화면과 글 읽기 편의]
-화면 테마 조정: 웹페이지의 색상 분위기를 부드러운 밝은 화면이나 어두운 화면으로 바꿔 부담을 줄일 수 있어요.
-화면 대비 줄이기: 너무 강한 색 대비와 이미지 자극을 낮춰 화면을 더 편안하게 볼 수 있어요.
-읽기 쉬운 글꼴: 본문, 버튼, 입력창 등에 더 읽기 쉬운 글꼴을 적용할 수 있어요.
-글자 크기 조절: 페이지의 주요 텍스트 크기를 80%에서 140% 사이로 조절할 수 있어요.
-줄 간격 조절: 문단과 목록의 줄 간격을 넓혀 글을 더 쉽게 따라갈 수 있어요.
-페이지 밀도 조절: 문단과 목록의 간격을 compact, normal, spacious 중에서 고를 수 있어요.
-자간 조절: 글자 사이 간격을 넓혀 글자를 더 또렷하게 구분할 수 있어요.
-단어 간격 조절: 단어 사이 간격을 넓혀 단어 단위로 더 쉽게 읽을 수 있어요.
-읽기 너비 조절: 본문 한 줄의 폭을 좁혀 시선 이동을 줄이고 읽기에 집중할 수 있어요.
-읽기 줄: 마우스 위치를 따라가는 가로 줄을 표시해 현재 읽는 위치를 놓치지 않게 도와줘요.

[집중과 읽기 도움]
-포커스 스포트라이트: 지금 읽는 줄이나 문단만 밝게 비추고 나머지는 어둡게 처리해 집중을 도와줘요. (화면 구조는 바꾸지 않아요)
-읽기 진행바: 기사 페이지에서 스크롤 위치를 기준으로 완료 비율과 예상 남은 읽기 시간을 보여줘요.
-텍스트 청킹: 긴 글의 문단 간격을 넓히고 구분선을 더해 작은 덩어리로 나눠 읽기 쉽게 만들어요.

[자극과 방해 요소 줄이기]
-움직임 줄이기: 애니메이션, 전환 효과, 부드러운 스크롤을 최소화하고, 움직이는 GIF는 첫 화면으로 멈추며, 영상 미리보기 같은 자동 움직임도 억제해요.
-자동재생 음소거: 자동으로 재생되는 미디어를 가능한 범위에서 멈추거나 음소거해요.
-이미지 부드럽게 보기: 이미지, 동영상, 일부 임베드 영상을 흐리게 보여주고 마우스를 올리거나 초점을 맞추면 선명하게 볼 수 있어요.
-광고 숨기기: 일부 광고 네트워크 요청을 차단하고, 페이지 안에서 광고처럼 보이는 영역을 접어 방해를 줄여요.
-광고 다시 보기: 접힌 광고 영역은 복구 버튼으로 다시 펼쳐볼 수 있어요.

[페이지 유형별 보조]
-리더 모드: 글 중심 기사 페이지를 자동으로 감지해(홈·포털·목록 페이지는 제외) 본문 폭과 주변 요소를 정리해 읽기에 집중하기 쉽게 만들어요.
-커뮤니티 보조: 커뮤니티 사이트 같은 복잡한 게시판이나 댓글 페이지의 간격과 가독성을 정리해요.

[사이트별 예외 설정]
-현재 사이트 예외: 특정 사이트에서만 확장 기능 끄기, 원래 색상 유지, 원래 글꼴 유지, 광고 허용 같은 예외를 설정할 수 있어요.
-사이트별 이미지 예외: 특정 사이트에서는 이미지 부드럽게 보기 기능만 따로 끌 수 있어요.
-사이트별 리더/커뮤니티 예외: 특정 사이트에서는 리더 모드나 커뮤니티 보조 기능만 따로 끌 수 있어요.

[AI 도우미]
-AI 도우미: OpenAI API 키나 자체 호스팅 백엔드를 설정하면 웹페이지 내용을 쉬운 말로 설명받을 수 있어요.
-선택한 텍스트 설명: 페이지에서 선택한 문장과 주변 맥락을 바탕으로 직접적인 의미, 의도, 다음 행동, 쉬운 표현을 알려줘요.
-현재 페이지 설명: 보이는 제목, 버튼, 주요 문단, 커뮤니티 항목을 바탕으로 페이지 목적과 주요 행동을 정리해줘요.
-폼 설명: 보이는 입력칸, 버튼, 경고 문구를 바탕으로 폼의 목적과 확인해야 할 내용을 설명해줘요.
-부드러운 AI 표현: AI 응답을 더 차분하고 직접적인 말투로 받을 수 있어요.
-AI 모델 선택: 팝업에서 사용할 OpenAI 모델을 선택할 수 있어요.

[AI 연결과 보호]
-API 키 저장: OpenAI API 키를 이 기기 확장 프로그램 저장소에 저장해서 직접 호출에 사용할 수 있어요.
-자체 호스팅 백엔드: API 키를 확장 프로그램에 저장하고 싶지 않을 때 로컬 백엔드 URL을 대신 사용할 수 있어요.
-민감한 페이지 보호: 로그인, 결제, 본인인증, 파일 업로드처럼 민감해 보이는 페이지에서는 AI 도움 기능을 제한해요.
-보이는 정보만 사용: AI 기능은 전체 HTML이나 입력한 값이 아니라 보이는 텍스트, 라벨, 버튼 같은 제한된 정보만 사용하도록 설계되어 있어요.
