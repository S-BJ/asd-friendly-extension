# ASD-Friendly UI 사용 안내

ASD-Friendly UI는 웹페이지를 조금 더 편하게 읽고 볼 수 있도록 돕는 Chrome 확장 프로그램입니다.  
모든 기능은 사용자가 직접 켜고 끌 수 있고, 사이트별로 예외 설정도 할 수 있어요.

## 설치 방법

1. ZIP 파일을 받은 경우 먼저 압축을 풀어주세요.
2. Chrome 주소창에 `chrome://extensions`를 입력해요.
3. 오른쪽 위의 `개발자 모드`를 켜요.
4. `압축해제된 확장 프로그램을 로드`를 누르고, 압축을 푼 확장 폴더를 선택해요.
5. 주소창 옆 확장 프로그램 아이콘을 눌러 필요한 기능을 켜면 됩니다.

## 주요 기능

- **글과 화면을 편하게 조정**  
  글자 크기, 줄 간격, 화면 밀도, 부드러운 색상 테마를 조정할 수 있어요.

- **이미지와 영상 썸네일 흐리게 보기**  
  이미지, 영상, 외부 영상 썸네일을 기본적으로 흐리게 보여줘요. 마우스를 올리거나 키보드로 포커스하면 다시 선명하게 볼 수 있어요.

- **광고와 방해 요소 줄이기**  
  일부 광고 네트워크 요청을 막고, 이미 보이는 광고처럼 보이는 영역은 접어서 화면 부담을 줄여요.

- **움직임과 자동 재생 줄이기**  
  불필요한 애니메이션을 줄이고, 자동 재생되는 미디어를 가능한 범위에서 멈추거나 음소거해요.

- **읽기 모드와 커뮤니티 보조**  
  긴 글 페이지나 댓글이 많은 커뮤니티 페이지에서 읽기 흐름을 조금 더 안정적으로 만들어줘요.

- **AI 도움 기능**  
  선택한 문장, 현재 페이지, 보이는 폼 구조를 쉬운 말로 설명해줘요. 이 기능은 사용자가 직접 실행할 때만 동작하고, OpenAI API 키 설정이 필요해요.

- **사이트별 예외 설정**  
  특정 사이트에서 원래 색상, 글꼴, 광고, 이미지 흐림, AI 도움 등을 따로 끌 수 있어요.

## 알아둘 점

- 로그인, 결제, 계정 설정, 건강, 법률, 신원 확인, 업로드처럼 중요한 페이지에서는 화면을 과하게 바꾸지 않도록 일부 기능이 자동으로 꺼질 수 있어요.
- 모든 광고나 모든 영상 썸네일을 완벽하게 처리하지는 못할 수 있어요. 사이트 구조가 바뀌면 동작이 달라질 수 있습니다.
- 이미지 흐림 기능은 이미지를 판단하거나 업로드하지 않아요. 브라우저 안에서 시각적으로만 흐리게 보여주는 기능입니다.
- AI 기능은 자동으로 실행되지 않아요. 사용자가 버튼을 눌렀을 때만 실행됩니다.
- 설정을 바꾼 뒤 바로 적용되지 않으면 페이지를 새로고침해보세요.
